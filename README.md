# Number.js
An insanely fast arbitrary precision library for JavaScript.

## Just *how* fast?

NumberJS takes only **`0.00006` seconds** per operation! That's over 16,000 operations per second!

## How do I use it?

1. Create an `Int` object
```js
var MyInt = new Int('23');
```
2. Do anything!
```js
MyInt.plus('32'); // 55
MyInt.times('4'); // 220
MyInt.pow('2');   // 48400
```
